## 如何使用

1. 克隆這個倉庫到本地端：
    ```sh
    git clone <倉庫網址>
    ```

2. 安裝必要的 Python 套件：
    ```sh
    pip install matplotlib jupyter
    ```

3. 啟動 Jupyter Notebook：
    ```sh
    jupyter notebook
    ```

4. 在瀏覽器中打開對應的 `.ipynb` 文件，按照教學內容進行學習和練習。
